Welcome to the official repository for the **Tech Conference 2025** website! This project is built using **React.js** and **Tailwind CSS** to create a modern, responsive, and visually appealing website for the conference.

---

## Features

- **Responsive Design**: The website is fully responsive and works seamlessly on all devices.
- **Dark Mode**: A toggleable dark mode for better user experience in low-light environments.
- **Multiple Pages**: 
  - **Home**: Hero section, featured speakers, and event highlights.
  - **Speakers**: Grid layout showcasing all speakers with their bios.
  - **Schedule**: Timeline of conference sessions.
  - **Sponsors**: List of sponsors categorized by tiers (Gold, Silver, Bronze).
  - **About**: Information about the conference and its purpose.
  - **Contact**: Contact form and social media links.
- **Dynamic Routing**: Built using `react-router-dom` for seamless navigation.
- **Tailwind CSS**: Utility-first CSS framework for rapid and consistent styling.
